# Security Policy

## Supported Versions

Latest `main` branch is supported.

## Reporting a Vulnerability

Please report security issues privately before public disclosure.

- Open a private security advisory on GitHub (preferred), or
- Email: security@your-domain.example (replace with real contact)

Include:
- affected version/commit
- reproduction steps
- impact assessment
- potential fix suggestions (optional)

We will acknowledge reports as quickly as possible and coordinate responsible disclosure.
