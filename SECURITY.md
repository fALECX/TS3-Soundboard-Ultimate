# Security Policy

## Supported Versions

The latest `main` branch is the supported release line unless a tagged release states otherwise.

## Reporting A Vulnerability

Report security-sensitive issues privately before opening a public issue.

Preferred contact path:

- Open a private GitHub security advisory for this repository

Fallback contact paths:

- X: https://x.com/fALECX
- Discord: `falecx`

Please include:

- affected version or commit
- reproduction steps
- impact assessment
- whether the issue is local-only, plugin-load related, or remotely triggerable
- any suggested mitigation if you already have one

Do not publish exploit details publicly until there has been a reasonable chance to investigate and fix the issue.
